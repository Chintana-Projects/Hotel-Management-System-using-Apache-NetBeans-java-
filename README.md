# Hotel Management System using Apache NetBeans JAVA
# (sem-3)

* In this project we use apache netbeans to execute the code, mysql workbench to store data.

* A person with valid credentials(username and password) to access the interface.

* The admin will be able to add employees, drivers and all customers.

* On the other hand the receptionist can also manage the hotel, which will contain options like :

     a. add rooms
  
     b. view departments
  
     c. view all employees
  
     d. view all customers
  
     e. access manager info
  
     f. add a new customer
  
     g. update status of customer(balance amount a customer has to pay)
  
     h. update room status(occupied/empty)
  
     i. pick-up service
  
     j. search room(based on availabily and if it'a a single/double bed)
  
     k. Check-Out
  
     l. Log out
